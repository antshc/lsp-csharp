== Install lsp setup skill
Works only for dotnet 10 sdk with the roslyn server
[source,bash]
----
gh skill install github/awesome-copilot lsp-setup
----
[source,json]
----
{
  "lspServers": {
    "csharp": {
      "command": "dotnet",
      "args": ["dnx", "roslyn-language-server", "--yes", "--prerelease", "--", "--stdio", "--autoLoadProjects"],
      "fileExtensions": {
        ".cs": "csharp"
      }
    }
  }
}
----


== Install OmniSharp LSP on Ubuntu 22.04
* For dotnet < 9 version install the .net using apt installation (DO not use snap)
* install standalone omnisharp
Note: not all lsp functions works. The incomingCalls, outgoingCalls, rename doesnt work.
[source,json]
----
{
    "lspServers": {
        "csharp": {
            "command": "omnisharp",
            "args": ["-lsp", "--stdio", "-s", "/home/dev/_sources/benefits/PaylocityBenefitsCalculator.sln"],
            "fileExtensions": {
                ".cs": "csharp"
            },
            "initializationTimeoutMs": 120000
        }
    }
}
----
